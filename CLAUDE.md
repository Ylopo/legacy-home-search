# Legacy Home Search — Project Rules

## What This Project Is
A branded real estate website for a new client. Built on the same Next.js + Sanity CMS stack as chris-nevada-next, but with a light modern design (white/cream backgrounds, Inter font, blue accent color) rather than the dark luxury style.

## Key Facts
- **Project name**: legacy-home-search
- **Sanity project ID**: `2nr7n3lm`
- **Sanity dataset**: `production`
- **Design theme**: Light modern — white/cream backgrounds, `#2563eb` blue accent, clean typography
- **Tech stack**: Next.js 16, Sanity CMS, Anthropic Claude, Vercel Cron, Upstash Redis, Resend email

## Sanity Configuration
All Sanity reads use project ID `2nr7n3lm` (not `r3saenct` which is the Chris Nevada project).

## Community Pages
All 6 community pages are live: `/virginia-beach`, `/chesapeake`, `/norfolk`, `/suffolk`, `/hampton`, `/newport-news`. Each page has a `CommunityBlogSection` that shows a scrollable Recent Posts carousel and a Most Popular pinned tab (see BLOG_PIPELINE.md → Community ↔ Blog Integration).

Update the `COMMUNITY_PAGES` array in `lib/assistant-tools.ts` if the AI Content Assistant needs to reference these pages.

## Content Machine Architecture

The full AI content pipeline — from research through social publish — runs entirely on this codebase. The pipeline: **Research → Idea Review → AI Writing → Fair Housing Check → Sanity → VA Queue (Thumbnail + Video + Captions) → Social Publish via Blotato → Bi-Weekly Performance Review**

Key lib files: `lib/research.ts`, `lib/idea-store.ts`, `lib/idea-writer.ts`, `lib/publish-service.ts`, `lib/fair-housing.ts`, `lib/heygen-client.ts`, `lib/blotato-client.ts`

- Full pipeline operations guide: **BLOG_PIPELINE.md**
- Fair Housing compliance: **FAIR_HOUSING.md**
- Voice/style intelligence (auto-updated weekly): **LEARNINGS.md**
- Multi-client replication guide + Shana Gates setup: **CONTENT_MACHINE_REPLICATION.md**

## Blog Pipeline
Full details in `BLOG_PIPELINE.md`. Two paths:
- **Idea Review (primary):** Daily cron researches Hampton Roads news → scored ideas queued in Redis → operator approves at `/admin/idea-review` → Claude writes post → Fair Housing check → Sanity `media_pending`
- **Blog Picker (fast-track):** Operator picks articles at `/admin/blog-picker/[date]` → same write + check + Sanity flow
- Both paths converge at the VA Queue where thumbnail, video, and captions are added before social publish
